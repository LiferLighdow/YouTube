## 🌟 Key Features

This project utilizes advanced **WebView injection technology** to deliver features that surpass the official application while maintaining a minimal footprint:

### 🎬 Multimedia Experience
- **Smart Ad-Blocking**: Deeply filters web-based advertisements for a pure viewing experience.
- **SponsorBlock Integration**: Automatically skips promotions, intros, and filler segments within videos.
- **Return YouTube Dislike**: Displays accurate dislike counts to help you judge content quality.
- **10x Playback Speed**: Break native limitations with support for up to 10x speed control.

### 🤖 AI Intelligent Assistant
- **Google Gemini Integration**: Generate one-click summaries, key timestamps, and deep analysis of video content.
- **Customizable Prompts**: Tailor the AI's focus based on your specific analytical needs.

### 📥 Powerful Download Manager
- **High-Quality Downloads**: Supports separate audio/video track grabbing with built-in **MediaMuxer** for lossless merging.
- **Asset Extraction**: Easily download high-resolution thumbnails and multi-language subtitles.

### 📱 System Integration
- **Seamless Background Play**: Audio continues playing even when the screen is off or you switch apps.
- **Picture-in-Picture (PiP)**: Intelligent PiP transitions for true multitasking.
- **Intuitive Gesture Controls**: Slide on the edges of the player to adjust volume and brightness.

---

## 🛠️ Technical Advantages

- **Privacy & Security**: All update tracking mechanisms have been completely removed. All JavaScript logic is stored locally in the `assets` directory, ensuring code execution is transparent and free from remote manipulation.
- **Minimalist Architecture**: Designed with almost zero internal dependencies, significantly reducing system resource consumption.
- **Adaptive Design**: Full support for Android 8.0+ adaptive icons and dark mode.

## 📦 Installation & Build

1. **Requirements**: Android 5.0 (API 21) or higher.
2. **Development Tools**: Android Studio Ladybug or newer recommended.
3. **Build Command**:
   ```bash
   ./gradlew assembleRelease
   ```
